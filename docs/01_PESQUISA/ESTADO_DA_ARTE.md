# 🏄 Eco Prancha — Estado da Arte

## 1. Processos Atuais de Fabricação de Pranchas

### 1.1 Estrutura Básica de uma Prancha

Toda prancha moderna compartilha a mesma anatomia:

```
BLANK (miolo espuma)
  ↕ Stringer (reforço central — madeira, carbono ou fibra)
  ↕ Fiberglass Cloth (tecido de fibra de vidro)
  ↕ Resina (poliéster ou epóxi)
  ↕ Acabamento (verniz / hot-coat)
  + Caixa de quilha + quilhas
```

### 1.2 Processo de Fabricação Passo a Passo

| Etapa | Descrição | Material |
|---|---|---|
| **1. Blank** | Bloco de espuma no formato bruto | PU, EPS, XPS ou PP |
| **2. Shaping** | Shaper modela o blank: raspa, lixa, dá rocker, outline, concaves | — |
| **3. Stringer** | Insere lâmina de madeira ou carbono no centro para rigidez | Balsa, cedro, carbono |
| **4. Laminação** | Fibra de vidro + resina sobre o blank | Fibra de vidro + resina poliéster/epóxi |
| **5. Hot-coat** | Camada fina de resina para selar | Resina + catalisador |
| **6. Lixamento** | Acabamento superficial | — |
| **7. Instalação** | Caixa de quilha + quilhas | Plástico, fibra |
| **8. Arte** | Pintura, verniz, logotipos | Tinta PU, verniz |

Referência: https://neptune.coach/blog/the-complete-guide-to-surfboard-construction-and-materials

### 1.3 Materiais Convencionais

| Material | Tipo | Característica | Usado em |
|---|---|---|---|
| **PU (poliuretano)** | Miolo | Tradicional desde anos 50. Flex progressivo, sensação clássica. Usa resina poliéster. | 60%+ das pranchas |
| **EPS (poliestireno expandido)** | Miolo | Mais leve que PU, usado com resina epóxi. Mais rápido, responsivo. "Skatey". | 30% das pranchas |
| **XPS (poliestireno extrudado)** | Miolo | Célula fechada, impermeável. Difícil de modelar. Usado em soft-tops (Wavestorm). | Iniciantes |
| **PP (polipropileno)** | Miolo | Novo padrão high-end. Resistente, flexível, leve. Usado em bodyboards profissionais. | Bodyboard, alta perf. |
| **EPP (polipropileno expandido)** | Miolo | Altamente resistente (passa carro por cima). Usado em bodyboards. | Bodyboard |
| **Poliéster** | Resina | Tradicional, tóxica, cheiro forte, cura rápida. Com PU. | PU boards |
| **Epóxi** | Resina | Mais forte, mais leve, menos tóxica que poliéster. Com EPS. | EPS boards |
| **Fibra de vidro** | Laminação | Tecido de vidro (E-glass) — 4oz, 6oz. Padrão industrial. | Todas as pranchas |

## 3. Impactos Ambientais e Contradições

### 3.1 Impactos da Indústria Convencional

| Impacto | Causa | Dado |
|---|---|---|
| **Toxicidade** | Resina poliéster — estireno, VOC | Altamente tóxico para shapers (danos neurológicos) |
| **Petroquímicos** | PU, EPS, PP, epóxi — todos derivados de petróleo | Fonte não renovável |
| **Resíduos** | Pranchas quebradas vão para aterro | Não biodegradáveis (séculos para decompor) |
| **Fibra de vidro** | Irritante dérmico e respiratório | Problemas de saúde ocupacional |
| **Transporte** | Pranchas importadas (China, Taiwan, EUA) | Pegada de carbono alta |

### 3.2 Contradições da Indústria

| Alegação | Realidade |
|---|---|
| "EPS é ecológico" | EPS é reciclável em tese, mas na prática quase nenhuma prancha é reciclada. Continua sendo poliestireno (petróleo). |
| "Epóxi é verde" | Epóxi é menos tóxico que poliéster, mas ainda é derivado de petróleo e não biodegradável. |
| "Prancha de madeira é sustentável" | Depende da origem da madeira. Paulownia cresce rápido, mas ainda é desmatamento. Resina epóxi continua. |
| "Impressão 3D é ecológica" | PLA é bioplástico, mas requer energia para impressão, e o descarte ainda é проблемático. |
| "Carbono é performance" | Carbono é petroquímico, caro, não renovável, difícil reciclar. |

### 3.3 Iniciativas "Eco" Existentes

| Marca | Proposta | Limitação |
|---|---|---|
| **Seawell** | PLA reciclado + impressão 3D | PLA é bioplástico, mas resistência? Escalabilidade? |
| **Kehl** | Bioespuma | Não encontrei detalhes da composição |
| **Siebert Corky** | Cortiça + EPS + madeira | EPS ainda é petroquímico. Cortiça é renovável, mas colada com epóxi? |
| **Index Krown ECO** | Bloco + resina + tecido sustentáveis | Mais completa, mas qual a real composição do bloco? |
| **Madeira (amigo Marcello)** | Paulownia | Resina epóxi ainda. Madeira tem limite de modelagem. |

## 4. Proposta Eco Prancha — Diferenciais

### 4.1 Comparação Direta

| Item | Convencional | Concorrentes Eco | **Eco Prancha (Takwara)** |
|---|---|---|---|
| **Miolo** | PU petroquímico | EPS reciclado, PLA | **PU Vegetal de Mamona** (Imperveg) |
| **Resina** | Poliéster / Epóxi | Epóxi (ainda petro) | **PU Vegetal Impermeabilizante** |
| **Fibra** | Fibra de vidro | Fibra de vidro | **Fibras vegetais** (juta, sisal, coco) |
| **Stringer** | Balsa / carbono | Balsa | **Bambu laminado** |
| **Acabamento** | Verniz PU sintético | Verniz PU sintético | **PU Vegetal** |
| **Toxicidade** | Altíssima | Média | **Baixíssima** (óleo de mamona) |
| **Biodegradabilidade** | Nula | Parcial | **Alta** (base vegetal) |
| **Fonte** | Petróleo | Mista | **100% renovável** |

### 4.2 Inovação Tecnológica Takwara

1. **PU Vegetal Expansivo** (nova fórmula Imperveg) — 200MPa de resistência, 50x mais leve que concreto. Potencial para miolo substituindo PU petroquímico.
2. **PU Vegetal Impermeabilizante** (UG 132A / MAMONEX RD70) — resina de laminação e acabamento, já comercializada.
3. **Compósitos vegetais** — estudos conclusivos de fibra de coco + PU mamona, fibra de sisal + PU mamona.
4. **Bambu laminado** — stringer e reforços, tecnologia consolidada Takwara (MPTDF).
5. **Flexibilidade sob demanda** — Imperveg e Purcom podem desenvolver formulações semi-flexíveis específicas para pranchas.

## 5. Frente de Atuação — Estratégia

### 5.1 Posicionamento

**Eco Prancha** não é mais uma "prancha ecológica" — é a **primeira prancha 100% vegetal do mundo**, com tecnologia integralmente nacional (Imperveg + Takwara).

### 5.2 Fases

| Fase | Produto | Parceiro | Validação |
|---|---|---|---|
| **1** | Bodyboard (bloco retangular PU vegetal) | Marcello + shaper amigo | Teste de flutuação, flexão, durabilidade |
| **2** | Prancha infantil / treino (molde) | Marcello + escola de surf | Uso supervisionado em escola |
| **3** | Prancha adulta completa | Marcello + shaper parceiro | Surfista avançado testa |
| **4** | Linha semi-artesanal | Fábrica parceira | Produção comercial |

### 5.3 Primeiros Testes (Fase 1)

- **Molde vs Bloco:** começar com **bloco retangular** de PU Vegetal Expansivo
- **Forma:** caixote de PVC (não gruda), desmolda, esculpe
- **Acabamento:** PU Vegetal Impermeabilizante por cima
- **Tamanho:** bodyboard (~1m) para facilitar manuseio e validar conceito
- **Local:** Limeira-SP (Fabio) ou oficina do shaper amigo do Marcello (RJ)
